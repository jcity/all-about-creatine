## Commission Priority Logic (for agent)

When writing an article, the agent should pick links in this order:

1. **Direct brand program** (10-20%) if available
2. **iHerb** (5-10%) if product is listed there
3. **Bodybuilding.com / Vitamin Shoppe** (5-8%)
4. **Amazon Associates** (3-5%) — universal fallback

Rule: *Always include at least one Amazon link as a fallback, even if a higher-commission option exists.*
